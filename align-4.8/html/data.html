Ontologies to compare

Note that there is a good source for ontologies: the famous Swoogle.

Here are mostly plans for providing ontologies to compare:

Available

A couple of ontologies onto1 and onto2 are available in the rdf directory for testing the current implementation.

Planned

Current plan includes describing several bibliographic ontologies in OWL in order to benchmark alignment.

  • format exmo: bib.dtd
  • KA2
  • VionDury
  • BibTeX
  • RAWeb
  • OpenOffice.org
  • BiblioML
  • DocBook

Resources

  • The Web-Based Knowledge Representation Repository featuring 47 test ontologies on the same domain.
  • The ontology translation source contains definitions for alignment results as well as some aligned ontologies.
  • The Illinois Semantic Integration Archive provides a number of data sets for testing alignment algorithms.
  • The pairs of the EON Ontology alignment contest contains 20 ontology pairs with reference alignments.
